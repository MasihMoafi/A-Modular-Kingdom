# Providers

Elpis runs in the terminal and supports the configured model provider for the local
session. On first launch, follow the interactive sign-in or configuration prompt;
credentials stay under the provider and local environment controls rather than being
placed in repository documentation.

## Choose deliberately

Use a provider configuration appropriate to the task's capability, cost, and network
requirements. Recheck a configuration when changing machines or switching providers:
environment variables, authentication state, and available models are mutable.

The [visual walkthrough](visual-walkthrough.md) covers the first-run flow, while
[context and sessions](context-and-sessions.md) explains what persists independently
of a provider session.
